# Zuk HM helper for Necromancy with Keybinds

Install by pasting https://drewbfg.github.io/afkwarden-presets/zuk-hm-helper-necro-keyed/dist/ in the alt1 browser and click "Add app",
or paste `alt1://addapp/https://drewbfg.github.io/afkwarden-presets/zuk-hm-helper-necro-keyed/dist/appconfig.json` in your browser.
